Awesome! Let’s make the first big upgrade. This part will add game statistics (games played, wins, win rate, and best time), and they will stay saved on the player’s device.

⸻

Part 1 – Statistics

1. Add new state

Find:

const [elapsed, setElapsed] = useState(0);
const [flagCount, setFlagCount] = useState(0);

Replace it with:

const [elapsed, setElapsed] = useState(0);
const [flagCount, setFlagCount] = useState(0);
const [stats, setStats] = useState(() => {
  const saved = localStorage.getItem("ms-stats");
  return saved
    ? JSON.parse(saved)
    : {
        games: 0,
        wins: 0,
        best: null,
      };
});

⸻

2. Save statistics

Add this below your other useEffect hooks:

useEffect(() => {
  localStorage.setItem("ms-stats", JSON.stringify(stats));
}, [stats]);

⸻

3. Count every new game

Inside your reset() function, add:

setStats((s) => ({
  ...s,
  games: s.games + 1,
}));

⸻

4. Save wins

Find:

if (checkWin(revealed)) setStatus("won");

Replace it with:

if (checkWin(revealed)) {
  setStatus("won");
  setStats((s) => ({
    games: s.games,
    wins: s.wins + 1,
    best:
      s.best === null || elapsed < s.best
        ? elapsed
        : s.best,
  }));
}

⸻

5. Add a statistics panel

Under the board, add:

<div className="stats">
  <h2>Statistics</h2>
  <p>Games: {stats.games}</p>
  <p>Wins: {stats.wins}</p>
  <p>
    Win rate:{" "}
    {stats.games === 0
      ? 0
      : Math.round((stats.wins / stats.games) * 100)}
    %
  </p>
  <p>
    Best Time:{" "}
    {stats.best === null ? "--" : stats.best + "s"}
  </p>
</div>

⸻
