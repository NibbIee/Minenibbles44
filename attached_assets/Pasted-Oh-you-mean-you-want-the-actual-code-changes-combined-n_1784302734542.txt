2. Replace your cell touch/click handlers

Change your cell <div> to:

<div
  key={`${r}-${c}`}
  className={cellClass}
  onClick={() => {
    if (cell.revealed) {
      handleChord(r, c);
    } else {
      handleReveal(r, c);
    }
  }}
  onContextMenu={(e) => handleFlag(e, r, c)}
  onTouchStart={() => {
    const timer = setTimeout(() => {
      handleFlag(
        { preventDefault: () => {} } as React.MouseEvent,
        r,
        c
      );
    }, 500);
    (window as any).flagTimer = timer;
  }}
  onTouchEnd={() => {
    clearTimeout((window as any).flagTimer);
  }}
>
  {content}
</div>

Now:

* Tap → reveal
* Hold → flag 🚩
* Right click → flag
